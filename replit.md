# ATOM.GAME - Система управления киберспортивными мероприятиями

## Обзор проекта
ATOM.GAME - веб-приложение для управления киберспортивными мероприятиями в атомных городах России. Поддерживает три роли пользователей: супер-администратор, организатор города и SMM-менеджер.

## Технологический стек
- **Frontend**: React + TypeScript + Vite
- **Backend**: Express.js + TypeScript
- **Database**: PostgreSQL с Drizzle ORM
- **UI**: Shadcn/ui + Tailwind CSS
- **State Management**: TanStack Query
- **Charts**: Recharts

## Структура проекта
```
├── client/src/
│   ├── components/        # UI компоненты
│   │   ├── dashboard/     # Дашборды для разных ролей
│   │   └── ui/            # Shadcn компоненты

```

## База данных
### Таблицы
- **users** - пользователи с ролями (super_admin, city_org, smm)
- **cities** - города с бюджетами
- **events** - киберспортивные мероприятия
- **media** - медиафайлы событий
- **expenses** - расходы
- **social_stats** - статистика соц.сетей
- **notifications** - уведомления

### Команды
```bash
npm run db:push    # Применить схему к БД
npx tsx server/seed.ts  # Заполнить БД тестовыми данными
```

## API Endpoints
- `GET/POST /api/users` - пользователи
- `GET/POST /api/cities` - города
- `GET/POST /api/events` - мероприятия
- `GET/POST /api/media` - медиафайлы
- `GET/POST /api/expenses` - расходы
- `GET/POST /api/social-stats` - статистика соц.сетей
- `GET/POST /api/notifications/:userId` - уведомления
- `GET /api/dashboard/stats` - статистика дашборда

## Роли пользователей
### SuperAdmin
- Просмотр всех городов и событий
- Утверждение календаря и бюджетов
- Управление пользователями
- Аналитика соц.сетей по всем городам

### CityOrg (Организатор города)
- Управление событиями своего города
- Работа с бюджетом
- Получение уведомлений

### SMM
- Просмотр ленты событий
- Работа с медиафайлами
- Отчёты по соц.сетям
